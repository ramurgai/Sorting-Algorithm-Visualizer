# Sorting-Algorithm-Visualizer

Name: Rahul Murgai

Project title: Sorting Algorithm Visualizer


Why is this a Java 2 Project?



This project will use many different sorting algorithms (some we've learned in this class and some we haven't) and allow the user to visualize their execution. 



What is your MVP?



The minimum viable product for this project will have multiple different sorting algorithms coded and a way for the user to see their execution through at least displaying the list after each step of the sort.


Stretch goals: Add some sort of UI so that the user can have fun looking through all the random sorting algorithms I use.


Solo or pair: Solo
